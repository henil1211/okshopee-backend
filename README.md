# ReferNex Monorepo

This repository has two apps:

- `backend/` Node.js API (deploy on Render)
- `frontend/` Vite React app (deploy build output to Hostinger)

Important:

- Backend root directory for Render must be `backend`.
- Frontend must be built from `frontend` and uploaded from `frontend/dist`.

Detailed setup instructions are inside:

- `backend/README.md`
- `frontend/README.md`
